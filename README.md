# Isograph Creation

## Overview
Isograph Creation is a project designed to help users create and manage isographs. An isograph is a visual representation of relationships between different components, which can be used in various fields such as data analysis, software engineering, and more.

## Features
- Create new isographs with various components.
- Visualize relationships and connections between components.
- Export isographs in multiple formats.
- User-friendly interface for managing and editing isographs.

## Installation
To install the Isograph Creation project, clone the repository:

```bash
git clone https://github.com/acridge3-369/isographcreation.git
```

Then navigate into the directory:

```bash
cd isographcreation
```
